# xAlert
JS script for Alerts

![](xAlertDemo.gif)

## Setup
To run this project, include the script like:

    <script src="xAlert.min.js"></script>
    
or 

    <script src="https://xi72yow.github.io/xAlert/xAlert.min.js"></script>
